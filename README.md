# Silence VScode
A vscode extension that turns off inline suggestions, code actions, validations, and all AI and autocomplete bloat. 

Just let me type my thing in peace man! 

# How to install

1. Download the vsix file.
2. Go to extensions
3. Install like this

# How to use 

1. Use the command pallete (default I think is ctrl+shift+p)
2. disabling will disable everything (backs up your original settings first) <br> <img width="903" height="165" alt="image" src="https://github.com/user-attachments/assets/d40b65ca-97d6-4b3f-8fb2-6fe5778379dd" />
